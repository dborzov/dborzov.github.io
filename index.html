Hi, my name is Dima, and I am currently a Ph.D. student in theoretical physics at UBC, Vancouver.

I like learning how things work and making new ones. Besides physics, I am interested in data analysis, machine learning and, more generally, using mathematics to understand the world better. I contribute to various open source projects, they are available at github.

Open source

Check out some of my open source projects:

  • Fredholm.py, is a Python library for solving Fredholm/integral equations numerically. Developed with problems in quantum many-body physics in mind, it is the only one, to the extent of my knowledge, that treats divergencies of principal value integrals properly.
  • subtitles2flashcards, indexing of movie/tv show subtitles for word frequencies and quotes.
  • Machine learning, algorithm implementations and homework solutions for the Caltech online course on machine learning.
